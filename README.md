This is the Tmux + VIM + zsh Install Progress.
=======================================

Tmux
----

1. Just copy the .tmux.conf to ~
2. install tmux.themepack

VIM
---

1. Install Vundle
2. Copy the .vimrc to ~
3. Open the vim and run the plugininstall

zsh
---
1. Install oh_my_zsh
2. Install powerline/fonts
3. Copy the .zshrc to ~
